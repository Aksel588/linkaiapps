LinkAI-Aion v0.1.2

linkai-aion is a lightweight Python utility library by LinkAI for text processing, file I/O, CLI parsing, and more. Designed for developers who want clean, reusable functions to speed up their workflow. Future versions will introduce AI features built on this solid foundation.

pip install linkai-aion==0.1.2
View on PyPI
Features
  • Text Utilities: count_words(), extract_emails(), summarize_text(), highlight_keywords()
  • File Handling: read_file(), write_file(), append_file()
  • Parser Tools: extract_numbers() from messy strings
  • and other features...
Why LinkAI-aion?
  • Clean, reusable functions for everyday Python tasks
  • Developer-focused, minimal dependencies
  • Future-proof: built to integrate with LinkAI APIs and models
  • MIT Licensed, open-source, Python 3.7+
Supported Python: 3.7+

Usage Example 
from aion.text import count_words, count_lines, summarize_text, extract_emails, extract_urls, highlight_keywords

sample_text = """
Hello, this is LinkAI-Aion!
Contact: aksel@example.com
Website: https://linkaiapps.com
This is a multi-line sample for testing.
"""

# Count words in the sample
print("Word count:", count_words(sample_text))

# Count lines in the sample
print("Line count:", count_lines(sample_text))

# Summarize the sample (first 2 lines)
print("Summary:", summarize_text(sample_text, max_lines=2))

# Extract emails
print("Emails:", extract_emails(sample_text))

# Extract URLs
print("URLs:", extract_urls(sample_text))

# Highlight keywords
print("Highlighted:", highlight_keywords(sample_text, ["LinkAI-Aion", "test"]))
